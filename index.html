Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A Python identifier must start with a letter; subsequent characters can also be digits (0-9) or underscores (_). Because Python is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).

Declaring Variables

Declaring variables are easy. First the variable name, equal sign, then the value

x = 42

Since Python is case-sensitive, myVar and myvar would be treated as separate variables.

You can use descriptive names for readability, and even Unicode characters are allowed for diverse identifiers. Avoid using Python’s reserved keywords, and keep your names meaningful, like user_age or total_sales.

Variable Scope

When you define a variable outside any function, it’s a global variable, accessible throughout the program. On the other hand, variables declared within a function are local variables, accessible only inside that function.

Python supports block-level scoping through constructs like if and loops. For example, variables created inside these blocks aren't automatically global. 

x = 5 # Global variable

def function():
    y = 2 # Local variable
Constants

Python doesn't have a native const keyword like some other languages. However, you can create constants by convention. This is done by naming variables in all uppercase letters, indicating their values should not change.

PI = 3.14

Unlike variables declared with const in JavaScript, Python constants can technically be reassigned, but this is discouraged by convention and good practice.

Data Types

Python has several built-in data types categorized into:

  • Numeric Types: int, float, complex for integers, decimals, and complex numbers.
  • Boolean Type: bool, representing True or False
  • Sequence Types: str for text, list for ordered collections, and tuple for immutable ordered collections.
  • Set Types: set and frozenset for unordered, unique collections.
  • Mapping Type: dict, a key-value pair structure.
  • Special Types: NoneType for None values.

Objects in Python hold data, while functions perform tasks or actions with that data.

If...else Statements

In Python, the if statement is used to execute code when a condition is true. The else statement provides an alternative for when the condition is false. Here's a basic example:

if condition:
    statement_1  # Runs if condition is true
else:
    statement_2  # Runs if condition is false

For multiple conditions, use elif to test them in sequence:

if condition_1:  
    statement_1
elif condition_2:
    statement_2
else:
    statement_last

Python evaluates conditions in order, and only the first true condition is executed. It's a good practice to group related code into blocks using consistent indentation. Avoid confusing assignment (=) with equality (==) in conditions.

While Statement

In Python, the while loop repeatedly executes a block of code as long as a specified condition is true. The basic structure looks like this:

while condition:
    statement
Function Declaration

In Python, a function is defined using the def keyword followed by the function name, a list of parameters inside parentheses, and a block of code indented underneath. Here's an example of defining a function that squares a number:

def square(number):
    return number * number

This function takes one parameter, number, and returns the square of that number. In Python, parameters are passed by reference for mutable objects and by value for immutable objects like integers. Changes to parameters within the function do not affect the original arguments unless explicitly returned or modified outside the function.
